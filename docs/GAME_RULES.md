- **CONFIG-001 — 라운드 설정**: 총 라운드는 10~300을 1 단위로 선택한다.
- **CONFIG-002 — 턴 제한**: 턴 제한은 15·30·45·60·75·90초 또는 무제한이다.
- **CONFIG-003 — 봇 실행 설정**: 봇 행동 지연과 빠른 시뮬레이션을 지원한다.

- **TURN-002 — 현재 플레이어**: 현재 플레이어만 턴 행동을 할 수 있다.
- **TURN-003 — 라운드 증가**: 마지막 활동 플레이어 턴 종료 시 global_round가 증가한다.
- **TURN-004 — 자기 턴 파산**: 자기 턴 파산은 현재 라운드 마지막 행동이며 다음 라운드는 남은 활동 인원으로 진행한다.
- **PAUSE-001 — 일시중지**: 일시중지 중 턴·봇·거래·승인 타이머와 무조작 판정이 모두 정지한다.
